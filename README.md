# Customer_Shopping_Behaviour_Analysis
Project Overview: Customer Shopping Behavior Analysis This project provides a data-driven analysis of 3,900 customer transactions to identify key spending patterns, customer segments, and product preferences. The goal of the analysis is to derive actionable business intelligence to guide strategic decision-making.
Data-Driven Insights: End-to-End Retail Performance Analytics📋 OverviewThis project delivers a comprehensive, end-to-end data analytics solution designed to transform raw transactional data into actionable business intelligence. The workflow covers the entire data lifecycle: programmatic ingestion and data cleaning via Python, robust relational storage and advanced querying using SQL, interactive dashboard development in Power BI, and executive-level reporting using Gamma App.The primary objective is to identify key sales drivers, optimize inventory turnover, and uncover high-value customer segments to support data-backed executive decision-making.📊 Dataset DescriptionName: Global Retail Sales & Customer TransactionsSource: E-Commerce & Retail Invoices DatasetSize: 3,900+ rowsKey Attributes:Transaction_ID (Unique identifier)Customer_ID (Unique customer token)Product_Category (Tech, Apparel, Home, etc.)Quantity, Unit_Price, DiscountOrder_Date, Ship_Status, Region🛠️ Tech Stack & ToolsData Processing & EDA: Python (pandas, numpy, matplotlib, seaborn)Database Management: PostgreSQL / MySQL / SQL ServerBusiness Intelligence: Power BI DesktopPresentation & Reporting: Gamma App, Markdown🚀 Project Workflow & Steps1. Data Ingestion & Exploratory Data Analysis (EDA)Automated data loading using Python.Executed descriptive statistics to analyze distributions, detect anomalies, and uncover initial correlations.Visualized outlier distributions in pricing and quantity variables.2. Interactive Power BI Dashboard DevelopmentImported data from the SQL Server into Power BI using an optimized schema.Developed custom DAX measures for advanced metrics.Designed an intuitive multi-page layout focusing on Executive Summary, Product Performance, and Regional Trends.5.
